# AI-Assisted PET Prep: A Parent's Methodology Guide

> A real-world experiment using AI to systematically guide a 9-year-old from KET to PET (Cambridge B1)  
> Includes: exam framework · prompt templates · bi-weekly reflections

---

### What This Project Is

Nova, age 9, Beijing. Has passed the Cambridge KET exam. Current goal: pass PET (B1).

This repository is not a study diary. It documents a **methodology** —
how to use AI to identify weak points, design practice paths, and iterate on learning strategy,
supported by a Notion-based knowledge management system.

If you are also preparing your child for a standardized English exam,
the frameworks and prompt templates here are designed to be directly reusable.

---

### What You Can Take From Here

- **Exam framework** — a full SOP from diagnostic to execution
- **Prompt templates** — for error analysis, weak-point identification, and study plan generation
- **Notion integration** — how to use a database to track learning progress
- **Bi-weekly reflections** — real progress updates and strategy adjustments
